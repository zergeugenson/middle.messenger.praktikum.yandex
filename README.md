# Мессенджер - чат

Минималистичный мессенджер, реализующий базовую функциональность: регистрация, авторизация, отсылка сообщений, редактирование профиля
Проект находится в стадии разработки


## Дизайн
Макеты находятся в папке ui проекта

## Используемые технологии:

- Vite
- Handlebars
- Express.js
- SCSS


## Установка
Установить зависимости проекта: `npm install`

Собрать проект: `npm run build`

Запустить проект на localhost: `npm run start`

Запустить проект в девелоперском режиме: `npm run dev`
