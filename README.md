# Мессенджер - чат

Минималистичный мессенджер, реализующий базовую функциональность: регистрация, авторизация, прием/отсылка сообщений, редактирование профиля.

## Дизайн
Макеты находятся в папке ui проекта.

## Используемые технологии:

- Vite
- Handlebars
- Express.js
- SCSS
- ESLint
- Stylelint
- Husky

## Установка
Установить зависимости проекта: `npm install`

Собрать проект: `npm run build` || `vite`

Запустить проект на localhost: `npm run start`

Запустить проект в девелоперском режиме: `npm run dev`

Запустить превью проекта: `vite preview`

Запустить проверку линтером: `npm run lint:fix`

Запустить проверку стилей: `npm run lint:style:fix`

Запустить тесты: `npm run test`
