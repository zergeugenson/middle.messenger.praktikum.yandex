# Мессенджер - чат

Минималистичный мессенджер, реализующий базовую функциональность: регистрация, авторизация, прием/отсылка сообщений, редактирование профиля.
Проект находится в стадии разработки.

## Дизайн
Макеты находятся в папке ui проекта.

## Используемые технологии:

- Vite
- Handlebars
- Express.js
- SCSS
- ESLint
- Stylelint
- Husky


## Странности

Использован пакет @hyperse/ts-node вместо ts-node, так как для удобства разработки в проекте использовался tsc-алиас '@=/src' для путей, а ts-node алиаcов не понимает, в связи с чем падал пакет mocha. Подробности см. https://github.com/justkey007/tsc-alias/issues/207

## Установка
Установить зависимости проекта: `npm install`

Собрать проект: `npm run build` || `vite`

Запустить проект на localhost: `npm run start`

Запустить проект в девелоперском режиме: `npm run dev`

Запустить превью проекта: `vite preview`

Запустить проверку линтером: `lint:fix`

Запустить проверку стилей: `lint:style:fix`
